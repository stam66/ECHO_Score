## Overview

**ECHOScore** delivers short looping echo videos per case and a structured checklist for learners to record findings. Admins manage users, cases, videos, and groups; the system supports OTP‑based password resets and SMTP email configuration.

### Default Admin (first run)

- **Username:** `admin`  
- **Password:** `<choose_a_secure_password>`  
> *(Set during initial deployment; credentials are not distributed in code.)*
