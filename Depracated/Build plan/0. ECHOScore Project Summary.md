# ECHOScore Project Summary
## Project Overview
ECHOScore is a web-based echocardiography training application built with XOJO Web that allows medical students to practice interpreting echo videos and receive immediate feedback.

## Core Functionality
- Students review echo case videos and complete assessments
- Case labels are hidden until after submission to prevent bias
- Videos display sequentially (one at a time with Previous/Next navigation)
- Color-coded feedback (green = correct, red = incorrect) after submission
- Automatic scoring based on 13 assessment criteria
- Draft saving capability to return to incomplete cases
- Admin tools for managing users, cases, and videos

## Architecture Decisions Made
### Database: MySQL
- 6 tables: users, cases, case_videos, user_responses, password_reset_tokens, email_config
- SHA2-256 password hashing
- Foreign key constraints with CASCADE delete

### XOJO Structure
#### Classes (3):
- wc_Base - Base container with positioning
- WebNavigationManager - Custom navigation system
- Session - Global state and database connection

#### Modules (2):
- EmailHelper - SMTP email sending and OTP generation
- PasswordResetHelper - Password reset token management
- DatabaseHelper - REMOVED (unnecessary, moved methods to Session)

#### WebContainers (10):
- Authentication: wc_Login, wc_ForgotPassword, wc_EnterOTP, wc_NewPassword
- User: wc_UserHome, wc_CaseReview
- Admin: wc_AdminHome, wc_UserAdmin, wc_CaseAdmin, wc_EmailConfig

#### WebPages (1):
- wp_MainShell - Single page with Placeholder for container embedding

#### Key Technical Decisions
- Custom navigation system - WebNavigationManager handles container switching with history
- Video storage - MP4 files in /Documents/CaseVideos/ folder
- Sequential video display - HTMLViewer with embedded video player, one video at a time
- Email via SMTP - Configurable through admin interface
- OTP expiration - 30 minutes
- Landscape layout - Optimized for desktop use

#### Video Display Implementation
- Videos loop automatically using JavaScript
- Embedded in HTMLViewer using relative paths: ../Documents/CaseVideos/filename.mp4
- Previous/Next buttons control which video displays
- Only one video visible at a time (not carousel/grid)

#### Assessment Criteria (13 items)
- LV Size - Dilated
- LV Function - Significantly Impaired
- RV Size - Dilated
- RV Function - Significantly Impaired
- Aortic Valve - Significant Stenosis
- Aortic Valve - Significant Regurgitation
- Mitral Valve - Significant Stenosis
- Mitral Valve - Significant Regurgitation
- Tricuspid Valve - Significant Stenosis
- Tricuspid Valve - Significant Regurgitation
- Pericardium - Significant Effusion
- IVC - High RA Pressure
- Requires Full ECHO?

#### Authentication Flow
- Login screen (username/password)
- Forgot Password → Email entry
- OTP sent via email (6-digit code)
- Enter OTP → Verify
- Create new password
- Return to login

#### User Workflow
- Login → Dashboard (list of cases)
- Select case → Opens wc_CaseReview
- Review videos using Previous/Next (case label hidden)
- Complete assessment checkboxes and conclusions
- Save Draft (can return later) OR Submit
- After Submit: Case label revealed, answers color-coded, score displayed, Submit button disabled

#### Admin Workflow
- Login → Admin Dashboard
- Manage Users: Add/Edit/Delete users
- Manage Cases: Add/Edit/Delete cases and their correct answers
- Add Videos: Upload videos with view descriptions and order
- Email Config: Configure SMTP settings for password reset emails
