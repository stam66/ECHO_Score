## Issue 1: Database Connection Failed
Symptoms: Error message on startup "Database connection failed"
**Solutions:**
- Verify MySQL is running: sudo systemctl status mysql
- Check credentials in Session.Opening
- Verify database exists: SHOW DATABASES;
- Check user permissions: GRANT ALL PRIVILEGES ON echoscore.* TO 'username'@'localhost';


## Issue 2: Videos Not Loading
Symptoms: "Your browser does not support the video tag" or blank video area
**Solutions:**
- Verify videos are in correct folder: /Documents/CaseVideos/
- Check video format (must be MP4)
- Verify video filenames match database entries exactly
- Check file permissions (should be readable by web server)
- Open browser console (F12) to check for 404 errors


# Issue 3: Email Not Sending
Symptoms: "Failed to send email" message during password reset
**Solutions:**
- Verify SMTP settings in email_config table
*For Gmail:
- Enable 2-factor authentication
- Generate app-specific password
- Use that password (not your Gmail password)*

- Check firewall allows port 587 (or 465 for SSL)
Review XOJO debug log for detailed error messages


## Issue 4: Password Reset OTP Not Working
Symptoms: "Invalid or expired OTP" message
**Solutions:**
- Verify email was received with correct OTP
- Check server time is correct: date
- OTP expires after 30 minutes - generate new one
- Verify password_reset_tokens table has entries


## Issue 5: Colors Not Showing After Submission
Symptoms: Checkboxes remain black instead of green/red
**Solutions:**
- Verify ShowCorrectAnswers() method is being called
- Check that correct answers exist in cases table
- Verify TextColor property is supported in your XOJO version
- Try using explicit color values instead of color constants


## Issue 6: Navigation Not Working
Symptoms: Containers don't appear or overlap incorrectly
**Solutions:**
- Verify all containers inherit from wc_Base
- Check Placeholder control has all Lock properties set to True
- Verify WebNavigationManager is instantiated in Session.Opening
- Check ContainerID and Position are set before NavigateTo()
